# Evidence Packs

Generate PDF evidence packs from Q&A conversations for negotiations and compliance.

---

## What are Evidence Packs?

Evidence packs are PDF documents that contain:
- Questions and answers from conversations
- Citations with page numbers
- Document excerpts
- Workspace and conversation metadata

Use evidence packs for:
- Contract negotiations
- Compliance documentation
- Legal reviews
- Stakeholder presentations

---

## Generate Evidence Pack

### From a Single Message

Generate evidence pack for a specific Q&A pair:

1. Navigate to **Q&A** page
2. Open a conversation
3. Find the assistant message with the answer
4. Click **Download Evidence Pack** (or use the export icon)
5. PDF downloads automatically

**Contents**:
- The question
- The answer
- All citations with page numbers
- Document excerpts
- Metadata (workspace, conversation title, date)

---

### From Entire Conversation

Generate evidence pack for all Q&A pairs in a conversation:

1. Navigate to **Q&A** page
2. Open a conversation
3. Click **Export Conversation** (in conversation header)
4. Select **Evidence Pack**
5. PDF downloads automatically

**Contents**:
- All questions and answers
- All citations
- Conversation summary
- Complete metadata

---

## Evidence Pack Contents

### Header Section

- Workspace name
- Conversation title
- Generation date
- Document count

### Q&A Sections

Each Q&A pair includes:

- **Question**: User's question
- **Answer**: Assistant's response with citations
- **Citations**: 
  - Document name
  - Page number
  - Section name
  - Text excerpt
  - Similarity score

### Footer

- Page numbers
- Generation timestamp

## Using Evidence Packs

### For Negotiations

1. Generate evidence pack for relevant Q&A
2. Share with legal team
3. Use citations to locate clauses in original documents
4. Reference in negotiation discussions

### For Compliance

1. Generate evidence packs for compliance-related questions
2. Include in compliance documentation
3. Maintain audit trail
4. Reference in reports

### For Reviews

1. Generate evidence packs for review sessions
2. Share with stakeholders
3. Use as meeting materials
4. Archive for future reference

## Best Practices

| Practice | Recommendation |
|----------|---------------|
| **Naming** | Use descriptive conversation titles |
| **Organization** | Group related questions in same conversation |
| **Citations** | Verify citations match original documents |
| **Archiving** | Save evidence packs with original documents |

---

## Troubleshooting

### Evidence Pack Won't Generate

- **Check conversation**: Ensure conversation has messages
- **Check citations**: Verify citations exist in messages
- **Check permissions**: Ensure you own the workspace

### Missing Citations

- **Verify documents**: Ensure documents are processed
- **Check question**: Rephrase if no relevant content found
- **Review search**: Check if documents contain relevant information
